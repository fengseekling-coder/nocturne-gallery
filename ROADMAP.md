# Roadmap

Gega Gallery is in an early public release. This roadmap tracks maintainer priorities and helps contributors understand where focused work is useful.

## 0.1.x: Public Repository Readiness

- Keep setup and verification documentation current.
- Add reproducible release notes and tags.
- Improve issue templates and contribution guidance.
- Document known platform-specific setup problems.

## 0.2.x: Import and Library Reliability

- Harden local file and folder import progress reporting.
- Improve duplicate confirmation UX and metadata.
- Expand thumbnail rebuild diagnostics.
- Add regression checks for group isolation and preview synchronization.

## 0.3.x: Packaging

- Stabilize macOS app packaging.
- Verify Windows packaging after migration cleanup.
- Add release artifact checks.
- Document platform-specific dependencies.

## 0.4.x: Maintainer Automation

- Add lightweight CI for TypeScript and Rust checks.
- Add focused tests for file path normalization.
- Add smoke tests for import and preview workflows.
- Improve media performance benchmark scripts.

## Longer-Term Work

- Better attachment preview coverage for design source files.
- Safer AI provider setup and model configuration.
- Search improvements across prompts, tags, metadata, and attachments.
- Accessibility review for keyboard navigation and contrast.
