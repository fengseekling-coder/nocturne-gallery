# Nocturne Gallery — 环境安装指南（精简版）

> 你只需要装 **一个 Claude Code CLI**，配好百炼 Key。
> Claude Code 会通过 Agent Teams 自动调度不同模型并行工作。
> 你全程只跟 Claude Code 一个窗口对话。

---

## Step 1：前置软件

| 软件 | 安装方式 |
|------|---------|
| Node.js v18+ | https://nodejs.org （LTS 版） |
| Git | https://git-scm.com |
| Rust | https://rustup.rs |

装完后打开 PowerShell 验证：

```powershell
node --version
git --version
rustc --version
```

---

## Step 2：安装 Claude Code

```powershell
npm install -g @anthropic-ai/claude-code
claude --version
```

---

## Step 3：获取百炼 Coding Plan Key

1. 打开 https://www.aliyun.com/benefit/scene/codingplan
2. 购买或确认你的套餐
3. 复制专属 API Key（格式 `sk-sp-xxxxx`）

---

## Step 4：配置 Claude Code 接入百炼

创建文件 `~/.claude/settings.json`（如果 `.claude` 目录不存在先创建）：

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "sk-sp-你的百炼Key",
    "ANTHROPIC_BASE_URL": "https://coding.dashscope.aliyuncs.com/apps/anthropic",
    "ANTHROPIC_MODEL": "qwen3.5-plus"
  }
}
```

创建文件 `~/.claude.json`：

```json
{
  "hasCompletedOnboarding": true
}
```

---

## Step 5：安装 Tauri 开发环境

```powershell
# VS Build Tools（如果没装过）
# 下载：https://visualstudio.microsoft.com/visual-cpp-build-tools/
# 安装时勾选 "Desktop development with C++"

# Tauri CLI
cargo install tauri-cli
```

---

## Step 6：验证

```powershell
# 启动 Claude Code
claude

# 进入后测试：
# 1. 输入 "你好" 看是否正常回复
# 2. 输入 /model 查看可用模型列表
# 3. 尝试切换：/model qwen3-coder-plus
# 4. 切回来：/model qwen3.5-plus
```

所有模型都能正常响应 = 环境搭建完成。

---

## Step 7：初始化项目

```powershell
# 进入项目目录
cd nocturne-gallery

# 安装依赖
npm install

# 启动开发服务器（纯前端预览）
npm run dev

# 启动 Tauri 开发模式（含 Rust 后端）
npm run tauri:dev
```

---

## 可用模型一览

通过百炼 Coding Plan，Claude Code 中可以 `/model` 切换以下所有模型：

| 模型 | 用途 | Claude Code 会自动在合适时机切换 |
|------|------|------|
| qwen3.5-plus | 综合推理、架构设计 | 主控默认模型 |
| qwen3-coder-plus | 编码主力 | Coder Teammate 使用 |
| qwen3-coder-next | 实验性编码 | CSS/样式任务 |
| kimi-k2.5 | 设计稿分析、长文档 | Analyst Teammate 使用 |
| glm-5 | 代码审查、测试 | Reviewer Teammate 使用 |
| MiniMax-M2.5 | 创意文案 | 文档撰写备用 |

你不需要手动切换，CLAUDE.md 里已经定义了规则，Claude Code 会自动分配。
